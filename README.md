# frontend

## Project setup
```
install node and npm
npm install -g yarn
sudo npm install -g @vue/cli
yarn install global @vue/cli
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
